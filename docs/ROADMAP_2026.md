# 🚀 Game Pipeline 2026 Vision: From Tool to Factory

지금까지 우리는 **"CLI 기반의 자동화 도구"**를 완성했습니다.
이제 우리는 이것을 **"누구나 사용할 수 있는 게임 생산 공장(Game Production Factory)"**으로 진화시켜야 합니다.

## 🏭 Phase 10: 웹 컨트롤 센터 (Web Control Center)
**"터미널(CLI)을 버리고, 웹에서 클릭 한 번으로 게임을 만듭니다."**

*   **대시보드 UI**: 현재 트렌드 키워드 시각화 (Google Trends 연동)
*   **GDD 편집기**: AI가 생성한 기획서를 웹에서 바로 수정하고 `Apply` 버튼 클릭
*   **빌드/배포 현황판**: Jenkins/GitHub Actions 스타일의 실시간 빌드 로그 확인
*   **통합 애널리틱스**: 모든 게임의 DAU/수익을 한 화면에서 모니터링
*   **기술 스택**: FastAPI (Backend) + Streamlit 또는 React (Frontend)

## 🧩 Phase 11: 장르의 다각화 (Multi-Genre Expansion)
**"러너 게임만 만드는 공장은 2류입니다."**

*   **Puzzle Template**: 매치-3 알고리즘, 콤보 시스템 코어 모듈화
*   **RPG Template**: 턴제 전투, 파티 관리, 장비 시스템
*   **Tycoon Template**: 방치형 자원 채굴, 업그레이드 수식 (BigInteger)
*   **공통 코어 확장**: `InventorySystem`, `QuestSystem` 등 장르 불문 공통 모듈 강화

## 🧠 Phase 12: Real AI Brain (LLM Integration)
**"Mock은 끝났습니다. 진짜 지능을 깨웁니다."**

*   **Gemini Pro Multimodal**: 텍스트 기획서뿐만 아니라, "이런 느낌의 스케치" 이미지를 보고 게임을 생성
*   **Code Generation**: 파라미터 조정을 넘어, `EnemyAI.gd` 같은 스크립트 코드를 직접 작성하여 주입
*   **Sound/Music gen AI**: 배경음악과 효과음까지 AI가 실시간 생성하여 에셋 파이프라인 완성

# 🚀 Game Pipeline: Beyond the Horizon

우리는 2D 하이퍼 캐주얼 게임 자동화의 정점에 도달했습니다.
이제 우리의 공장은 **차원(Dimension)**과 **연결(Connection)**의 벽을 넘어야 합니다.

## 🧊 Phase 13: 차원 도약 (Dimension Leap)
**"2D Pixel을 넘어 3D Low-Poly의 세계로."**

*   **3D Runner Template**: Temple Run 스타일의 3D 무한 러너 구현.
*   **Blender Automation**: Python 스크립트를 사용하여 간단한 3D 모델(나무, 장애물) 자동 생성.
*   **GridMap System**: 3D 타일맵 기반의 레벨 생성 자동화.

## 🌏 Phase 14: 멀티버스 (Multiplayer Universe)
**"혼자 하는 게임은 외롭습니다. 전 세계와 연결합니다."**

*   **Dedicated Server**: Godot High-level Multiplayer API를 활용한 서버/클라이언트 구조.
*   **Matchmaking**: Nakama 또는 직접 구현한 매치메이킹 서버 연동.
*   **AI Bot**: 플레이어가 부족할 때 투입될 똑똑한 AI Agent 구현.

## 🗣️ Phase 15: 목소리의 힘 (Voice Command)
**"키보드 없이 말로 게임을 만들고, 말로 플레이합니다."**

*   **Whisper Integration**: "점프력이 너무 낮아, 좀 더 높여줘"라고 말하면 GDD 파서가 즉시 `jump_force` 수정.
*   **Voice Gameplay**: "점프!", "공격!" 음성 인식 게임 플레이 모드.
